TOKEN COUNTER — PROPOSALS

1. ARC GAUGES
Semicircular gauges for input/output tokens. Color changes as usage grows. Glow on active.
2. SEGMENTED BARS
Segmented bars showing breakdown: cached vs new tokens. Compact, fits current panel.
3. CONTEXT RING
Single ring showing context usage. Inner text shows %. Segments for system/tools/messages.
4. METRIC CARDS
Compact stacked cards with mini sparklines. Shows trend over time.
5. RADIAL HUD
JARVIS-style radial display with concentric rings. Sci-fi aesthetic matching the orb.

Arc Reactor with Embedded Metrics

How should metrics integrate into the arc reactor?

A. Rings as Gauges

Each concentric ring is a gauge: outer=queue, middle=response time, inner=actors. Core shows status + uptime. Arcs fill proportionally. Labels between rings. Clean and data-dense.

B. Pure Reactor + Side Metrics

Arc reactor stays purely visual (status color + animation). Metrics live in a separate text panel beside it. Cleaner separation — reactor is identity, panel is data.

C. Reactor + Floating Labels

Arc reactor visual with metric labels floating around it at cardinal points (top/right/bottom/left). Connected by thin lines. Feels like a tactical HUD targeting system.
